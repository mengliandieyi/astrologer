# 行旅筹划·灵犀行程 - 功能规格书

## 项目概述

**模块名称：** 行旅筹划·灵犀行程  
**定位：** 智能旅行规划助手，AI 一站式规划行程、预算、清单  
**核心价值：** 用户只需输入日期和偏好，AI 自动生成完整旅行方案

---

## 用户输入

### 必填
- 📅 出行日期（开始日期、结束日期）

### 选填
- 💰 预算范围（如：5000元以内、1-2万）
- 👥 出行人数（1-10人）
- 🎯 偏好标签（可多选）：
  - 海边度假 / 山景自然 / 城市观光 / 美食之旅 / 亲子游 / 浪漫之旅 / 探险挑战 / 文化历史 / 购物休闲 / 温泉养生
- 🚗 交通偏好（飞机 / 高铁 / 自驾 / 火车）
- 🏨 住宿偏好（经济型 / 舒适型 / 高端酒店 / 民宿）
- 📍 出发地（用于计算交通费用）

---

## AI 自动生成内容

### 1. 目的地推荐
- 根据日期季节、预算、偏好推荐 3 个目的地
- 每个目的地显示：
  - 🏷️ 名称 + 简介
  - 🌤️ 出行日期天气概览
  - 💰 参考预算
  - 📸 2-3 张代表性图片（网络图片 URL）
  - ⭐ 推荐理由
- 支持"换一批"重新推荐

### 2. 每日行程安排
用户选择目的地后，生成每日行程：
- 📍 每日景点列表（含游览时长）
- 🚶 景点间路线规划
- 🚗 交通方式 + 耗时
- 🍜 推荐餐饮（含人均消费）
- 🏨 住宿推荐（含价格区间）
- 💡 当地小贴士

### 3. 预算拆分
- 总预算估算
- 详细拆分：
  - ✈️ 交通（往返机票/火车票）
  - 🏨 住宿（酒店总价）
  - 🍜 餐饮（每日估算）
  - 🎫 门票/体验项目
  - 🛍️ 购物/其他
- 每人分摊费用（如多人出行）
- 💵 建议换汇金额（如出境）

### 4. 行李清单
根据目的地、天气、天数自动生成：
- 👔 衣物类（按天数+天气推荐）
- 💄 日用品类
- 💊 药品类（常备药）
- 📱 电子设备
- 📄 证件类
- 🎒 特殊物品（如防晒、登山装备等）

支持勾选完成、添加自定义物品

### 5. 注意事项
- 🛂 签证/通行证要求
- 🌧️ 天气提醒
- ⚡ 电压/插头信息（如出境）
- 🗣️ 当地语言常用语（10句）
- 💱 支付方式（现金/刷卡/移动支付）
- 🚨 安全提醒
- 📶 电话卡/流量建议

## 实时信息接入

### 天气（必接）
- 未来 7-15 天天气预报
- 出行日期内的具体天气
- 建议携带物品（雨伞、防晒等）

### 景点/酒店信息（可选）
- 景点门票价格
- 酒店价格区间
- 航班价格参考

## 技术实现

### 前端
- 框架：React + TypeScript（参考 BaziPage.tsx）
- 样式：Tailwind CSS + 现有 design system
- 路由：React Router（/travel）
- 状态管理：React hooks
- AI 输出：流式展示（SSE 或轮询）

### 后端
- 框架：Express + TypeScript
- API 设计：
  - `POST /api/travel/recommend` - 获取目的地推荐
  - `POST /api/travel/generate` - 生成完整行程
  - `POST /api/travel/weather` - 获取天气信息
  - `POST /api/travel/adjust` - 调整行程
  - `GET /api/travel/save` - 保存行程
  - `GET /api/travel/list` - 获取行程列表
  - `POST /api/travel/share` - 生成分享卡

### AI 调用
- 复用项目现有的 AI 调用逻辑（参考 /api/reports/ai）
- 使用流式输出改善体验
- Prompt 工程：针对旅行场景优化

### 数据存储
- 行程数据存 PostgreSQL
- 用户收藏/历史存 PostgreSQL
- 缓存热门目的地信息
